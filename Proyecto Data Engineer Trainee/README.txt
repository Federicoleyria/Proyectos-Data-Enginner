Proyecto Data Enginner Trainee


Este código es un script de Python que convierte un archivo de texto con formato TSV 
(valores separados por tabulaciones) en un archivo CSV (valores separados por comas), 
permitiendo especificar el delimitador de salida.



Este mini Proyecto fue realizado através de la página https://www.knowledgehut.com/blog/data-science/data-engineering-projects
con documentación y codigo fuente para su realización.
